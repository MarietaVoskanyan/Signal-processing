# Signal Sampling and Interpolation

This project demonstrates **signal sampling** and **interpolation** techniques using Python, NumPy, Matplotlib, and SciPy. It covers the effects of different sampling rates on sinusoidal signals and shows how to restore sampled signals using interpolation.

## Features
- Generate sinusoidal signals with different frequencies.
- Sample signals at multiple sampling rates and visualize the effects.
- Demonstrate the **Nyquist theorem** and aliasing using under-sampled signals.
- Restore sampled signals using **cubic interpolation**.
- Handle multi-component signals (e.g., combination of 3 Hz and 15 Hz sine waves).

## Dependencies
- Python 3.x
- NumPy
- Matplotlib
- SciPy
